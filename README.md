# Yassine Jenjare - Portfolio Website

A modern portfolio website built with React and TypeScript to showcase my skills and projects as a Full-Stack Software Engineer.

## Features

- **Modern Design** - Clean and professional look with Tailwind CSS
- **Responsive** - Works great on all devices and screen sizes
- **Project Showcase** - Display your best work and projects
- **Skills Section** - Highlight your technical skills
- **Experience** - Share your work history and experience
- **Contact Form** - Let visitors reach out to you

## Getting Started

### Prerequisites
You need Node.js installed on your computer.

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and go to the URL shown in the terminal (usually `http://localhost:5173`)

## Available Commands

- `npm run dev` - Start development server
- `npm run build` - Build the project for production
- `npm run lint` - Check code quality

## Technologies Used

- React 19
- TypeScript
- Tailwind CSS
- Vite
- React Router
